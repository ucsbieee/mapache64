Working on an Issue

This project is organized into issues:

Arcade Machine Issue List

If you want to contribute, follow these steps:

Steps

  1. Find something interesting on the issue list that hasn’t been started yet.
  2. Create a new branch with a branch name related to the issue.
  3. Add your GitHub username and your new branch’s name to the issue’s page.
  4. Move the issue to the “branched” column on the projects page.
  5. Complete the issue on your branch, while editing as few lines/files as you can.
  6. TEST YOUR CODE!!!
  7. Submit pull request.
  8. Follow up with code reviewers if they find any issues.
