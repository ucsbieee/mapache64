Floating Point vs. Fixed Point

The 65C02 does not have any built-in floating-point functionality. Therefore, we will instead use fixed-point numbers for continuous values.

Fixed Point Numbers

We have a Q11_4 implimentation in firmware.js. You can see its usage in game-demo.js.

Never use floating point numbers! If you need non-integers, use Q11_4.

Filling the Pattern Memory

Pattern memory stores the pixel data for the sprites (PMF) and tiles (PMB). Pattern Memory is divided into blocks of 16 bytes, where one block is one sprite/tile. Observe Ethan’s Binary Hacking Class notes:

Sprites
  • 11 is white
  • 10 is light gray
  • 01 is light dark
  • 00 is transparent/black
