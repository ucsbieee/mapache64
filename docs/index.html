Mapache 64

Custom 6502 Video Game Console

Description

Mapache 64 is is a custom video game console based around the 6502 microprocessor. This console uses a custom graphics processing unit (GPU) written in SystemVerilog and runs on a Cmod A7-35T. The firmware and games are programmed using C, and can be compiled with a custom linker script and cc65, an open-source C to 6502 assembly compiler. This project was developed by UC Santa Barbara's IEEE chapter, lead by Ethan Sifferman.
